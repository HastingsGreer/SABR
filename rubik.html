Rubik's Cube



Solving a fully or partially scrambled Rubik's cube is another very hard problem. However, the setup is simple and follows the same form as the machinarium puzzle.

Board indicates all of the sides, Start indicates the scramble, and End indicates the solution. Trans operates over a side object and can be clockwise or counter clockwise. The side object has a DesObj for each Rubik's Cube side.

It has been proven via intelligent brute-force that at most 20 moves are needed to solve a Rubik's Cube from any start position. So the number of stages input parameter can always be below 20.
